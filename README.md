# learn_getx

A Taskflow mobile, it's call API to https://github.com/nxhung2304/taskflow repository.

# Getting started
## 1. fvm
```
brew tap leoafarias/fvm
brew install fvm
fvm install 3.29.3
fvm use 3.29.3
```

## 2. Get dependencies
```
fvm flutter pub get
```

### 3. Running the project
```
fvm flutter run
```
